# Virtual composer
This is a simple and primitive program
to compose various compositions
without of user's intervention.

## Requirements
* python2
* sox

## Install and run
### Install
    git clone https://github.com/donarturo11/virtcomposer.git
### Run
    chmod +x virtcomposer.py
    ./virtcomposer.py
